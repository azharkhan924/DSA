1. Two Sum 

Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].

Approach: Put target-nums[i], i  as key,value if not already present in map

class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer,Integer> m=new HashMap<>();

        for(int i=0;i<nums.length;i++){
            if(m.containsKey(nums[i]))return new int[]{m.get(nums[i]),i};
            else {
                m.put(target-nums[i],i);
            }
        }
        return new int[]{-1};
    }
}

2. Sort Colors

Input: nums = [2,0,2,1,1,0]
Output: [0,0,1,1,2,2]

Approach 1: Count the number of 0,1,2 and insert manually

class Solution {
    public void sortColors(int[] nums) {
        int count0=0,count1=0,count2=0;
        for(int i:nums){
            if(i==0)count0++;
            else if(i==1)count1++;else if(i==2)count2++;
        }
        for(int i=0;i<nums.length;i++){
            if(i<count0)nums[i]=0;
            else if(i<count0+count1)nums[i]=1;
            else nums[i]=2;
        }
    }
}

Approach 2: using dutch flag algo
take low mid as 0 and high as n-1
if nums[mid] is 0 swap nums[low] with nums[mid] and increment both low and mid
if nums[mid] is 1 then increment mid
if nums[mid] is 2 then swap nums[mid] with nums[hight] decrement high

class Solution {
    private void swap(int nums[],int i,int j){
        int t=nums[i];
        nums[i]=nums[j];
        nums[j]=t;
    }
    public void sortColors(int[] nums) {
     int low=0,mid=0,high=nums.length-1;
     while(high>=mid){
        if(nums[mid]==0)swap(nums,low++,mid++);
        else if(nums[mid]==1)mid++;
        else swap(nums,mid,high--);
     }
}
}


3. Majority Element

The majority element is the element that appears more than ⌊n / 2⌋ times. You may assume that the majority element always exists in the array.

Approach 1: Use HashMap count frequency of each element and return element with frequency>n/2

class Solution {
    public int majorityElement(int[] nums) {
        Map<Integer,Integer> m=new HashMap<>();
       for(int i:nums){
        m.put(i,m.getOrDefault(i,0)+1);
        if(m.get(i)>nums.length/2)return i;
       }
       return -1;
    }
}

Approach 2: Boyer-Moore Voting Algorithm

The majority element appears:

- more than n / 2 times.
- its frequency is greater than all other elements combined.
So if we:

cancel one majority element
with
one non-majority element

the majority element will STILL survive.

class Solution {
    public int majorityElement(int[] nums) {
        int element=nums[0],count=0;
        for(int i=0;i<nums.length;i++){
            if(count==0)element=nums[i];
            if(element==nums[i])count++;
            else count--;
            
        }
        return element;
    }
}

4. Maximum Subarray

Input: nums = [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
Explanation: The subarray [4,-1,2,1] has the largest sum 6.

Approach: Kadane Algorithm
get sum of elements while traversing linearly if sum gets negative store the maxsum and intialise sum with 0

class Solution {
    public int maxSubArray(int[] nums) {
        int sum=0,maxsum=Integer.MIN_VALUE;

        for(int i=0;i<nums.length;i++){
            sum+=nums[i];
            maxsum=Math.max(maxsum,sum);
            if(sum<=0)sum=0;
        }  
        return maxsum;     
}
}
